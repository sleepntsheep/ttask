tui task manager built in python
<hr>

## usages

key
```
a: add new task
r: remove selected task
j, k/ up, down: navigate selected task
d: mark task as done 
w: write changes to todo.txt
q: quit and write changes
e: edit priority
```

## installation
```
pip3 install ttask
```

## dependencies

windows:
>  windows-curses
